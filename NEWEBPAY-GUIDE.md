# 藍新金流（NewebPay）申請與串接手冊 — Guide.Ferryman

適用版本：網站 v4.9｜金流模式：信用卡定期定額（月費／年費自動扣款）

---

## 全流程總覽（約 1～2 週）

```
① 註冊個人會員（10分鐘） → ② 身分與銀行驗證（3~5工作天）
→ ③ 建立商店 → ④ 申請開通信用卡支付＋定期定額（人工審核2~3工作天）
→ ⑤ 建立 4 條定期定額委託連結 → ⑥ 貼進網站 GF_PAY → 上線收款
```

## ① 註冊個人會員

1. 前往 **www.newebpay.com** → 右上「註冊」→ 選 **「個人會員」→ 免費註冊**（不需公司登記）。
2. 填基本資料（姓名須與身分證、銀行帳戶一致）、設定帳號密碼、勾選同意條款。
3. 收註冊完成通知信 → 登入後台。

## ② 身分與銀行驗證

登入後台 → **會員中心 → 基本資料設定**，依序完成：

| 驗證項目 | 準備物 | 時間 |
|---|---|---|
| 身分驗證 | 身分證正反面照片＋健保卡照片 | 上傳後審核約 1~3 工作天 |
| Email／手機驗證 | 收驗證碼 | 即時 |
| 金融機構帳號 | 你要收款入帳的銀行帳戶（存摺封面） | 銀行驗證約 5 工作天 |

> 💡 審核通過**不一定會寄信**，隔幾天主動登入後台看狀態。

## ③ 建立商店

**會員中心 → 商店管理 → 建立商店**，建議填法：

- 商店中文名稱：`Guide.Ferryman 產業顧問工作室`
- 商店英文名稱：`Guide Ferryman Strategic Advisory`
- 商店網址：`https://falconkenny.github.io/gf-website/`
- 商品類型：顧問服務／數位內容訂閱

## ④ 申請開通「信用卡支付」（含定期定額）

**商店管理 → 詳細資料 → 申請信用卡支付**，勾選「**信用卡一次付清**」與「**信用卡定期定額**」送出。人工審核約 2~3 工作天，審核人員會實際查看你的官網。

**過審重點（你的網站大多已具備）**：品牌與服務介紹 ✅、明確定價（會員方案浮框）✅、聯絡方式 ✅。**建議補一項**：頁尾加「服務條款／退費政策」說明（訂閱制商品審核常看這個）——你說一聲，我 10 分鐘幫你生一頁。

若被退件：看 Email 中的失敗原因補件重送即可，審核是真人，資訊補齊就會過。

## ⑤ 建立 4 條定期定額委託連結

開通後：後台 **信用卡定期定額管理 → 新增委託（連結）**，建立以下 4 條：

| 名稱 | 金額 | 週期 | 授權期數 |
|---|---|---|---|
| 白銀會員・月費 | NT$3,200 | 每 1 個月 | 不限次數（客戶可隨時停） |
| 白銀會員・年費（半價） | NT$19,200 | 每 12 個月 | 不限次數 |
| 黃金會員・月費 | NT$6,000 | 每 1 個月 | 不限次數 |
| 黃金會員・年費（半價） | NT$36,000 | 每 12 個月 | 不限次數 |

每條建立後會得到一個網址（客戶點開＝輸入卡號授權，之後每期自動扣款）。

## ⑥ 貼進網站

打開 `assets/js/membership.js` 最上方，把 4 條連結貼進對應欄位：

```js
const GF_PAY = {
  silver_twd:      "貼_白銀月費連結",
  gold_twd:        "貼_黃金月費連結",
  silver_twd_year: "貼_白銀年費連結",
  gold_twd_year:   "貼_黃金年費連結",
  ...
```

存檔上傳 GitHub → 升級浮框會自動出現「台幣付款：藍新信用卡定期定額」按鈕，並依客戶選的「月費/年費」自動切換對應連結。

## 日常營運 SOP（收款 → 開通 → 續扣）

1. 客戶在網站送出升級申請 → 你收到 Formspree 通知信、後台出現「待付款」。
2. 客戶點付款按鈕 → 在藍新頁面完成信用卡授權 → **藍新寄「授權成功」通知信給你**。
3. 後台「⭐ 升級申請與通行碼」→ 收款確認・開通 → 繳費方式選「信用卡定期定額」（會自動標記 🔁 自動續費）→ 寄通行碼信給客戶。
4. 之後每期藍新自動扣款並寄通知信給你 → 到後台按該客戶的「⏫ 續期」延長效期。
5. 客戶要取消：藍新後台終止該筆委託 ＋ 網站後台將通行碼「停用」。
6. 每天 08:30 的到期檢查信會提醒你誰快到期、誰已自動停權，不會漏。

**提領**：款項先進你的藍新帳戶，再依藍新公告的撥款天數與提領方式轉到你驗證過的銀行帳戶。

## 費用與額度備忘

- 免月費、免設定費；信用卡手續費依官網公告（約 2.8%，經銷通路有優惠費率方案）
- 個人會員信用卡額度：30 天內 20 萬（黃金月費可容納約 33 位）；用到 8 成藍新會通知，可申請調升
- 無營業登記可收款，但屬個人所得需申報；銷售額成長後（勞務月銷 4 萬即有稅籍登記義務）請洽會計師

## 下一階段（拿到金鑰後找我）

商店建立後，後台「商店管理 → 詳細資料」可取得 **商店代號、HashKey、HashIV** 三組串接金鑰。把這三組**私下**提供給我（存入 GitHub Secrets，絕不放進網頁程式碼），我就能建 Supabase Edge Function 接藍新的扣款通知（NotifyURL），做到：**付款成功自動發通行碼、每期扣款自動續期、扣款失敗自動提醒**——屆時整套訂閱收費就是全自動，你只要看報表。
